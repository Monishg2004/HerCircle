1. **Clone the Repository**
   ```bash
   git clone https://github.com/Monishg2004/HerCycle.git
   cd HerCycle
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
